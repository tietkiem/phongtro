Công Cụ Cập Nhật Website

Sau khi đã thêm tất cả các tin đăng mới, nhấn nút bên dưới để tải về file website đã được cập nhật.

  1. Nhấn nút "Tải về file website đã cập nhật".
  2. Một file index.html mới sẽ được tải về máy tính.
  3. Trong thư mục dự án, xóa file `index.html` cũ và thay thế bằng file vừa tải về.
  4. Đăng (push) toàn bộ thư mục dự án lên GitHub để hoàn tất.
